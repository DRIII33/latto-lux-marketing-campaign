# **Latto Lux: A Data-Driven Marketing Campaign**

## **Analyst & Creator:** Daniel Rodriguez III

## Project Overview

This repository houses the end-to-end data science project for the "Latto Lux" marketing campaign. The campaign's centerpiece is a physical scratch-off lottery ticket, which serves as a novelty product and a gateway to a digital, interactive app experience.

Our primary objective is to leverage data analytics and machine learning to optimize marketing spend, understand customer behavior, and drive over $1 million in revenue within the first 6 to 12 months. This project is a collaboration between the marketing team, the Latto Lux development team, and the data science team, all working to create a highly successful and data-informed product launch.

## Key Objectives

* **Campaign Performance Tracking:** Build a real-time dashboard to monitor key metrics such as sales, app downloads, and user engagement.
* **Audience Segmentation:** Use data to identify different user personas and tailor marketing messages for maximum impact.
* **Predictive Analytics:** Develop models to forecast sales and anticipate inventory needs.
* **ROI Measurement:** Accurately attribute revenue to specific marketing channels to ensure effective budget allocation.

## Repository Structure

* `data/`: Contains raw and processed CSV files used for analysis.
* `notebooks/`: Includes Google Colab notebooks for data generation, cleaning, and modeling.
* `reports/`: Houses reports and presentations for stakeholders and team members.
* `data_dictionary.md`: A comprehensive guide to all data fields used in the project.
* `README.md`: This document.
